## 2. `while` ‚Äî –ø–æ–≤—Ç–æ—Ä—è—Ç—å, –ø–æ–∫–∞ —É—Å–ª–æ–≤–∏–µ –≤–µ—Ä–Ω–æ

### üí° –ü—Ä–∏–º–µ—Ä: –∏–≥—Ä–∞ ¬´–£–≥–∞–¥–∞–π —á–∏—Å–ª–æ¬ª

```python

    –ø–æ–ø—ã—Ç–∫–∏ += 1

    if –ø–æ–ø—ã—Ç–∫–∞ == –∑–∞–≥–∞–¥–∞–Ω–Ω–æ–µ:
        print(f"üéâ –ü–æ–∑–¥—Ä–∞–≤–ª—è—é! –£–≥–∞–¥–∞–ª –∑–∞ {–ø–æ–ø—ã—Ç–∫–∏} –ø–æ–ø—ã—Ç–æ–∫!")
        break  # ‚Üê –Ω–µ–º–µ–¥–ª–µ–Ω–Ω—ã–π –≤—ã—Ö–æ–¥ –∏–∑ —Ü–∏–∫–ª–∞
```

## 3. –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Ü–∏–∫–ª–æ–º: `break`, `continue`, `pass`

| –ö–æ–º–∞–Ω–¥–∞ | –ß—Ç–æ –¥–µ–ª–∞–µ—Ç |
|--------|------------|
| `break` | **–°—Ä–∞–∑—É –≤—ã–π—Ç–∏** –∏–∑ —Ü–∏–∫–ª–∞ (–¥–∞–∂–µ –µ—Å–ª–∏ —É—Å–ª–æ–≤–∏–µ –µ—â—ë –∏—Å—Ç–∏–Ω–Ω–æ) |

| `pass` | ¬´–ó–∞–≥–ª—É—à–∫–∞¬ª ‚Äî –Ω–∏—á–µ–≥–æ –Ω–µ –¥–µ–ª–∞–µ—Ç, –Ω–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç –Ω–µ –æ—Å—Ç–∞–≤–ª—è—Ç—å –ø—É—Å—Ç–æ–π –±–ª–æ–∫ |

### üí° –ü—Ä–∏–º–µ—Ä—ã –≤ –∫–æ–¥–µ

```python

# üîπ continue: –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ–º —Ç–æ–ª—å–∫–æ –ø–æ–ª–æ–∂–∏—Ç–µ–ª—å–Ω—ã–µ —Ç–µ–º–ø–µ—Ä–∞—Ç—É—Ä—ã

    print(f"+{t}¬∞C")
# ‚Üí +22¬∞C, +33¬∞C, +10¬∞C, +1¬∞C, +23¬∞C


# üîπ pass: –∑–∞–≥–ª—É—à–∫–∞ (—Ä–µ–¥–∫–æ –Ω—É–∂–Ω–æ, –Ω–æ –±—ã–≤–∞–µ—Ç)

        pass  # ‚Üê –≤—Ä–µ–º–µ–Ω–Ω–æ –Ω–∏—á–µ–≥–æ –Ω–µ –¥–µ–ª–∞–µ–º ‚Äî –∫–æ–¥ –Ω–µ —Å–ª–æ–º–∞–µ—Ç—Å—è

# ‚Üí –®–∞–≥ 0 –≤—ã–ø–æ–ª–Ω–µ–Ω
# ‚Üí –®–∞–≥ 1 –≤—ã–ø–æ–ª–Ω–µ–Ω
# ‚Üí –®–∞–≥ 2 –≤—ã–ø–æ–ª–Ω–µ–Ω
```

## 4. `zip()` ‚Äî ¬´–º–æ–ª–Ω–∏—è¬ª –¥–ª—è –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö —Å–ø–∏—Å–∫–æ–≤
